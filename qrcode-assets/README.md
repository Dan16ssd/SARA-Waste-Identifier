# SARA QR Code Assets

This directory contains permanent, high-quality static QR codes pointing to the SARA application.

## Destination URL
- **URL**: `https://sarascan.zeabur.app/`
- **Purpose**: Direct scan link to the Material Identifier scanner page of the SARA application.

## Files Included

1. **`sarascan-qr.svg`**: Scalable Vector Graphics format.
   - *Best for*: Printing on flyers, banners, physical trash bins, or resizing to any scale without losing quality.
   - *Error Correction*: High (Level H, allows up to 30% damage/obstruction).
2. **`sarascan-qr.png`**: Portable Network Graphics raster image.
   - *Best for*: Digital displays, websites, emails, or mobile applications.
   - *Dimensions*: 1024x1024 pixels (high resolution).
   - *Error Correction*: High (Level H, allows up to 30% damage/obstruction).

## Why Static QR Codes?
Static QR codes encode the destination URL directly into the matrix. They never expire, do not require any redirect servers or middleman accounts, and will remain permanent as long as the destination URL `https://sarascan.zeabur.app/` is online.

## Technical Specifications
- **Error Correction**: Level H (~30% recovery capacity)
- **Data Encoded**: `https://sarascan.zeabur.app/`
